## What This Is

A **local browser dashboard** that gives the desk one window with:

- **Left:** Koyfin Whinfell workspace
- **Right:** Barchart futures / basis view
- **Bottom:** 5-minute workflow, 6 saved prompts, Gross Risk inputs

Workaround for Comet split-screen limitations. No server required.

## First-Time Setup (2 minutes)

1. **Paste your Koyfin URL** in the left pane header  
   - Use your saved Whinfell workspace link from Koyfin (e.g. `https://app.koyfin.com/...`)
2. **Confirm Barchart URL** in the right pane  
   - Default: Major Commodities futures page  
   - Or paste a custom watchlist / `CL`, `ES` quote page
3. Click **Save Settings** — URLs and Gross Risk fields persist in browser local storage
4. If panes show a fallback message, click **Open ↗** — some sites block embedding; use side-by-side tabs with bottom utility panel still active

## Daily Workflow (< 5 minutes)

| Step | Action | Where |
|------|--------|-------|
| 1 | Read Transmission State + Whinfell Score | Koyfin (left) |
| 2 | Scan credit / rates | Koyfin left column |
| 3 | Scan futures leadership + basis | Barchart (right) |
| 4 | Copy Prompts ① + ② to your agent | Bottom → Prompts tab |
| 5 | Update Gross Risk + posture | Bottom → Gross Risk tab |

## Bottom Panel Tabs

| Tab | Purpose |

| **6 Saved Prompts** | Copy-paste ready agentic prompts |

| **Quick Tickers** | One-click Barchart quotes (C3 master list subset) |

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Blank iframe | Click **Open ↗** — site blocks embed; use new tab + keep dashboard open |
| Settings lost | Same browser profile required; click **Save Settings** after changes |
| Wrong Koyfin page | Paste correct workspace URL → **Reload Panes** |
