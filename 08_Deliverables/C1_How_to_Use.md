## Quick Start (~2 minutes)

1. Open `C1_Whinfell_Credit_Confirmation_Score_Logic.md`
2. Go to **Section 6** — score each of the 8 components (Bullish / Bearish / Mixed / Neutral)
3. Start at **50**, add/subtract weights per **Section 3**
4. Clamp result to 0–100
5. Map to interpretation band in **Section 7** and apply positioning guidance
